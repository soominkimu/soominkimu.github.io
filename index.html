SpacetimeQ Labs - Experiments on the Web

TopoJSON + React - Earth Globe and Map Projections

GeoJSON and TopoJSON parser implemented with React with no dependency to any thrid party libraries. Tween animation between map projections.

WebGL + React - Planet Earth

3D Earth globe with texture mapping, WebGL/Three.js/React. Another option for the realistic earth globe.

Canvas + React - SQ3D Demo

SQ3D: JavaScript/ES6 3D engine, no dependencies. 1007 stars visible in the zodiac constellations. Wireframes demo with the heatmap of Tokyo in 2018.

Canvas + React - SQ3D Polygons

SQ3D: JavaScript/ES6 3D engine, no dependencies. Searching for the geometrical beauty of the calendar/time - 4 seasons, 12 months, 24 divisions, 72 sub-divisions, 365 days.

SVG + React - ArtMoon

SVG and JavaScript frame animation test. Quasi-3D expression with SVG and filters. Phases of the Moon can be drawn without any 3D library.

Canvas + React - Heatmap of Tokyo, 140 years

Rendering to HTML5 Canvas, data from 1876 to 2019. Converted from CVS to JSON by a Python program.

HTML5/CSS + React - Calendar with Heatmap

CSS Layouts with React. Calendar test the real data from the Japanese Meteorological Agency. Fetch API test: JSON server required.

SVG + React - SpacetimeQ Prototype #1

SVG (Scalable Vector Graphics) is so powerful! The first experiment to check the potential of the SVG with React. Initial prototype of the SpacetimeQ that integrates watch, calendar, and other utilities.

CSS/SASS - Front End Experiment

How much detail in the layout can you control with CSS? A Japanese tabloid cover page is a good example of the complexity of layout. Guess which one is the web version of the magazine cover.
